# H1B Visa Applications Dashboard (2025)
H1B Visa Applications Data Analysis using SQL and Power BI

This project analyzes H1B visa application data using SQL and Power BI to uncover insights about employer activity, salary trends, and job roles.

## Tools Used
- MySQL
- Power BI
- Data Cleaning
- Data Visualization

## Dataset
The dataset contains over 2.4 million H1B visa application records.

## Key Insights
- Total H1B Applications: 2.4M+
- Average Salary: $129K
- Top Employers Sponsoring H1B
- Most Common Job Roles
- Salary Distribution of H1B Jobs
- State-wise job opportunities

## Dashboard Features
- Interactive filters
- KPI cards
- Map visualization
- Employer analysis
- Job role insights
- Salary trends
